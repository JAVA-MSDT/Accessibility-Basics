Checkboxes Group recommendations

Checkboxes Container:

  • Use role=“group”, to announce the content as groups of element
  • Use aria-labelledby=“customLabelId“, if you have a custom label
  • Use aria-label=“describe the group“, if you have no custom label

Checkboxes List if you use UL:

  • Give it a role=“presentation or none”, so by this way Firefox will list all the menu items in array

Checkboxes :

  • Use “for” and “id” attributes to attach labels to form elements
  • No need to add role attribute If you already added a type attribute

Radios Group recommendations

Radios Container:

  • Use role="radiogroup", to announce the content as radio group elements
  • Use aria-labelledby=“customLabelId“, if you have a custom label
  • Use aria-label=“describe the group“, if you have no custom label

Radios List if you use UL:

  • Give it a role=“presentation or none”, so by this way Firefox will list all the menu items in array

Radios :

  • Use “for” and “id” attributes to attach labels to form elements
  • No need to add role attribute If you already added a type attribute
