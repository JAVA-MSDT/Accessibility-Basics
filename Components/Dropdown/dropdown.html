Standard Dropdown Menu Accessibility

Standard Drop Down menu recommendations

Menu Button:

  • Give it an aria-haspopup="menu", to indicates that the pop up is a menu
  • Give it an aria-controls="menuList-ID“, List which will be opened
  • Give it an aria-expanded="false“, It will be a true when it is opened

Menu UL

  • Give it a role="menu"
  • Give it an aria-labelledby=“MenuButtonId"

Menu List if there is one

  • Give it a role=“presentation or none”, so we can have the correct numbers of menu elements

Menu items

  • Give it a role="menuitem"
  • If you are using button, it will be focusable and keyboard accessible by default.
  • If you are using anchor tag, give it href=“” to make it keyboard accessible
