Tab Accessibility

Tab controls (Tabs Links & contents)

Tabs Container { UL or DIV }

  • Give it a role of tablist, role=“tablist”

Tabs Buttons

  • Give it a role of tab , role=“tab”
  • Give it an aria-controls pointer which point to the tab panel id, aria-controls=“tabpanelId"
  • Give it an aria-selected=“false", it will be changed automatically when the tab being selected,

Tabs List, if you use UL as container of tab buttons

  • Give it a role=“presentation or none”, so we can have the correct numbers of tabs button's

Tabs Panels

  • Give it a role=“tabpanel”
  • Give it an aria-labelledby=“tabButtonId“, which point to the tabbutton id of tabpanel ,
