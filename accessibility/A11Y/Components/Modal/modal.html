Modal Accessibility

Modal dialog recommendations

Modal Container:

  • If you are using div element as a container git it a role="dialog"
  • aria-modal ="true“, to identify it as a modal
  • aria-labelledby=“dialog_label-id”, Point to a title inside the modal, to announce a title of the modal on open
  • aria-describedby=“dialog_message-id”, Point to a message inside the modal, to announce it when the modal opens such removing or confirmation message

Dynamic content recommendations

Dynamic content alets, Spinner , etc:

  • Use role=“role of the content”, to announce the new information
  • Use aria-live="polite“ in case this is not a critical information
